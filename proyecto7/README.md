# Análisis de anuncios de venta de coches

Esta aplicación web permite explorar visualmente un conjunto de datos de anuncios de venta de coches (vehicles_us.csv). Proporciona dos funcionalidades principales:

- **Histograma**: muestra la distribución del odómetro (kilometraje) de los vehículos.
- **Gráfico de dispersión**: relaciona el odómetro con el precio de venta.

Los gráficos se generan mediante botones (o casillas de verificación) y se visualizan de forma interactiva con Plotly. La aplicación está construida con Streamlit.
